== Description ==

With the **wezzoo** plugin you have a map, focused on the region you want, displaying the **realtime weather** conditions.
No forecast! Just the true weather coming from the community.
And **your own community** (your visitors) can be part of it by posting their weather observations.
wezzoo plugin is both a tool giving you local weather information and engaging your audience.

https://www.youtube.com/watch?v=BtW3ZmiS-mM

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wezzoowidget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('wezzoowidget_hook'); ?>` in your templates

== Frequently Asked Questions ==

= When is the content updated ? =

Everytime there is a new content it appears on the map.
And content displayed is no older than 1 hour.
